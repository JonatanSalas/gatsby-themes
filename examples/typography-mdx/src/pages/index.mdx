# Gatsby Theme Typography MDX

Use [Typography.js][] with [MDX][].
This example uses [typography-mdx][] to add Typography.js styles to MDX documents without adding global styles.
The typography-mdx package create components for MDX based on a Typography.js theme, using [Emotion][] for scoped styles.
These styles will only apply to MDX documents, and will not affect HTML elements created outside of an MDX component.

## Altering the theme

To change the theme, you'll need to shadow the theme's typography module. Create a directory at `src/gatsby-theme-typography-mdx` and add a `typography.js` module.

```js
// src/gatsby-theme-typography-mdx/typography.js
export { default } from 'typography-theme-grand-view'
```

## Webfonts

If the typography theme you decide to use requires webfonts, use the
`gatsby-plugin-google-fonts` plugin.

```js
// gatsby-config.js
module.exports = {
  __experimentalThemes: [
    'gatsby-theme-typography-mdx',
  ],
  plugins: [
    {
      // example for typography-theme-grand-view
      resolve: 'gatsby-plugin-google-fonts',
      options: {
        fonts: [
          'montserrat:700',
          'arvo:400,400i,700',
        ]
      }
    },
  ]
}
```
