
# Getting Started

Install the theme as a dependency in your Gatsby site.

```sh
npm i gatsby-theme-sidebar
```

Add the theme to your `gatsby-config.js`.

```js
// gatsby-config.js
module.exports = {
  __experimentalThemes: [
    'gatsby-theme-sidebar',
  ],
}
```

Create a directory for shadowing the theme components.

```sh
mkdir src/gatsby-theme-sidebar
```

Add a `Root` component to configure the theme's layout.

```jsx
// src/gatsby-theme-sidebar/root.js
import React from 'react'

export default ({
  Layout,
  Header,
  Main,
  Sidebar,
  Content,
  Footer,
  children,
}) =>
  <Layout>
    <Header>
      Header content
    </Header>
    <Main>
      <Sidebar>
        Sidebar content
      </Sidebar>
      <Content>
        {children}
      </Content>
    </Main>
    <Footer>
      Footer content
    </Footer>
  </Layout>
```

Each component in the layout can be styled using Emotion's `css` prop.

```jsx
// example styles
<Sidebar
  css={{
    backgroundColor: '#f6f6f6',
  }}>
  Sidebar content
</Sidebar>
```
