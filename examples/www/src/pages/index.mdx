
# Gatsby Themes

A work-in-progress exploration of Gatsby themes

https://github.com/jxnblk/gatsby-themes

## System Gatsby Theme

A minimalist Gatsby theme based on constraint-based design principles, built with [MDX][], [Emotion][], and [Styled System][].

```sh
npm i gatsby-theme-system
```
