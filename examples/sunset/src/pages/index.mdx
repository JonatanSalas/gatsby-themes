## Notes

- theme extensions
  - sunset can import and alter components from sidebar
  - end user imports sunset components and composes a layout
    - with content added in JSX
  - sunset shadows sidebar/root and adds styles
    - but end user cant shadow sunset/root without removing changes

  }}>
